Projects

[Unity] 3D Statue Puzzle

We were inspired to create a 3D Statue Puzzle game because we wanted to implement photogrammetry into our game. The models of the used statue were imported into blender and split into pieces. In this Unity game, the player controls the rotation of the camera using the “a” and “d” keys. The player can also access an image of the completed statue using the “m” key. The goal is to drag pieces of a broken statue back together through three levels. This game was created for Global Game Jam 2020.

Collaborated with Spencer Poisseroux and Mehraj Meah.

[Python] BrickHack V 2019

We were inspired to create a Visual Novel because we wanted to parody the common tropes of other visual novel games. This game was developed in a game engine called Ren’Py. The game engine allowed us to work with 2D animation and UI elements to let us create a Visual Novel that allowed for branching paths for different endings. This game was created for BrickHack V.

Collaborated with Sarah Strickman and Kristy Lo.
